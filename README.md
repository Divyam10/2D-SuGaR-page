# 2D-SuGaR Project Page

GitHub Pages site for the paper **"2D-SuGaR: Surface-Aware Gaussian Splatting for Geometrically Accurate Mesh Reconstruction"** (Eurographics 2026 Short Papers).

## Setup

1. Copy the `assets/` folder from the main repo into this directory:
   ```bash
   cp -r ../2d-Sugar_push/assets ./assets
   ```

2. Push this folder to a GitHub repo and enable GitHub Pages (Settings → Pages → Deploy from branch `main`, root `/`).

The site will be live at `https://<your-username>.github.io/<repo-name>/`.
